A User need to login to use the page somehow.

A loggedin User can create multiple Scenarios, and will see them in a Overview.

A User can invite other Perople to this Scenario using a invite link.

A Person do not need to be logged in to play Scenario.
But a logged in User can also join Scenarios, which will appear in on Overview.






class User < ApplicationRecord
  has_many :scenarios
  has_many :joinings
  has_many :joined_scenarios, through: :joinings, source: :scenario
end

class Scenario < ApplicationRecord
  belongs_to :user
  has_many :joinings
  has_many :users, through: :joinings
end

class Joining < ApplicationRecord
  belongs_to :user
  belongs_to :scenario
  has_many :results
end

class Result < ApplicationRecord
  belongs_to :joining
end

rails generate model user
rails generate model scenario user:references
rails generate model joining user:references scenario:references
rails generate model result joining:references
